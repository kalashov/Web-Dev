# Online Store (Lab 4, Task 2)

Angular app with a product catalog and share functionality.

## Run locally

```bash
npm install
ng serve
```

Then open `http://localhost:4200/`.

## Features

- 10 products with real `kaspi.kz` product links
- `ProductList` and `ProductCard` standalone components
- Responsive grid layout
- Product image gallery (thumbnail switcher)
- Share buttons for WhatsApp and Telegram using encoded URLs
